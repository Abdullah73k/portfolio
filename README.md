# Portfolio

My personal portfolio website showcasing my projects, work experience, and skills.

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Run the development server:

```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Magic UI
